Overview

This project is a high-end, luxury landing page for Mercedes-Benz, designed to capture leads for potential customers. The primary goal was to replicate the brand's sophisticated aesthetic while ensuring a seamless user experience.

The layout features a split-screen design: a dynamic Bootstrap carousel showcasing premier models and a clean, conversion-focused contact form tailored for both desktop and mobile devices.

Built With

  • HTML5 & CSS3: Built with CSS Variables and Flexbox for a maintainable and modern structure.
  • Bootstrap 5: Implemented the Carousel component with fade transitions and responsive grid utilities.
  • Mobile-First Workflow: Fully responsive design using Media Queries to adapt the layout from dual-column to a single-column stack.
  • Google Fonts: Integrated 'Heebo' to ensure clean, modern typography across all languages.

What I Learned

This project provided great insights into professional UI/UX development:

  • RTL Support: Mastering Right-to-Left layout logic while maintaining alignment and visual hierarchy.
  • Media Handling: Using object-fit: cover and CSS scaling to ensure high-quality car imagery remains proportional on any screen size.
  • Luxury UI Aesthetics: Learning how to use dark themes, subtle shadows, and specific color palettes to convey a premium brand feel.

Continued Development

Future enhancements planned for this project include:

  • Advanced Validation: Adding real-time JavaScript form validation for phone numbers and email formats.
  • Micro-interactions: Implementing GSAP or CSS animations for subtle "scroll-reveal" effects on the form elements.
  • Backend Integration: Connecting the form to a server-side script or service like Netlify Forms for actual data collection.

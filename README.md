## üéì What I Learned
Developing this central hub sharpened my skills in professional UI/UX:

* **Brand Identity:** Creating a cohesive look and feel using a specific color palette (Navy & Green) and consistent typography.
* **Component-Based Logic:** Building reusable CSS classes and structures for project cards and "fancy lists."
* **Form Integration:** Implementing `Getform.io` to handle server-side logic without a dedicated backend, ensuring a reliable contact method.
* **Interactive Elements:** Mastering CSS transitions and `backdrop-filter` effects for a premium, high-tech feel.
